# RentingAll - Rent Anything, Anytime

A peer-to-peer vehicle rental marketplace that connects vehicle owners with people who need them.

## About the Project

RentingAll is a visionary early-stage startup that allows users to rent various vehicles directly from owners. The platform supports:

- Renting a wide range of vehicles (bikes, cars, etc.) in one app
- Direct owner-to-renter transactions
- Competitive and affordable prices
- Community-driven marketplace

## Getting Started

### Prerequisites

- Node.js 16.8 or later
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/renting-all.git
cd renting-all
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Features

- Modern, responsive UI built with Next.js and Tailwind CSS
- Full-stack application with API routes
- Authentication system
- Vehicle listings with search and filter capabilities
- Booking system
- Messaging between renters and owners
- Payment processing
